JavaScriptwas developed in 1995, and the official name is ECMA ( European Computer Manufacturers Association). It is a OOP language, meaning is is an object oriented program. Not to be confused with Java. There are a five major components of JavaScript, they are:

  • Operators
  • Variables
  • Functions
  • Methods
  • Data Types

Operators are symbols (addition, subtraction, etc) used to denote operations within functions.

Variables can be thought of as a container to store information

Variable naming conventions should always be named in a way that are easy for other developers to understand (i.e. what it is for, or what it is doing). They are case sensitive and so, they should always be named using the the lowerCamelCase method.

Functions it is a block of code used specifically for JavaScript. It sets the parameter for code to be excuted.

Method is an action ran on a piece of data. Furthermore, the method can be thought of as the message. And should not be confused with the response, which is the return.

Argument is the local variable available within all fuctions. It always begins with the value 0, but can be modified to other values to manipulate the outcome of the response.

Parameters are the names listed in the function definition.

Return is what the function outputs after recieving the method, and argument from the parameters.

Chaining methods is combining multiple methods to the response of a previous method, or variable.

String is text that is always inside of quotes, you can use single or double.

Boolean always returns true or false

Undefined specifies a currently undefinded value.

NaN is an undefined number, such as: 0%0

Escape is represented by a \ in order to cancel out any formatting or symbols.

Alert is a method used to create a pop-up the user interacts with containing information.

Comments can be made using 'alt + /' to leave notes for youself or other developers. Can also be used to temporarily disable lines of codes.

jQuery is a library of JavaScript.

Attribute is the tag of html.
